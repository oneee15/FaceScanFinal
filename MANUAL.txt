Run app:
 terminal 1 (CMD): npx react-native start --reset-cache

Clean cache:
 terminal 3 (Powershell): cd android, ./gradlew clean, cd ..

Build APK:
 terminal (Powershell): cd android, ./gradlew clean, ./gradlew assembleRelease
